# Guide de Démarrage - PMS

Bienvenue sur votre plateforme de gestion immobilière !

## Premiers Pas

### 1. Ajouter une Propriété
1. Allez dans **Propriétés** → **Ajouter**
2. Remplissez les informations (nom, adresse, type)
3. Ajoutez les unités (appartements, bureaux, etc.)

### 2. Enregistrer un Locataire
1. Allez dans **Locataires** → **Ajouter**
2. Entrez les coordonnées du locataire
3. Le locataire sera disponible pour les baux

### 3. Créer un Bail
1. Allez dans **Baux** → **Nouveau bail**
2. Sélectionnez le locataire et l'unité
3. Définissez les dates et le montant du loyer

### 4. Suivre les Paiements
1. Allez dans **Tableau de bord** pour voir le résumé
2. Les paiements en attente sont affichés en rouge
3. Marquez les paiements comme reçus

## Fonctionnalités

| Fonction | Description |
|----------|-------------|
| **Propriétés** | Gestion des bâtiments et unités |
| **Locataires** | Profils et contacts |
| **Baux** | Contrats avec dates et montants |
| **Paiements** | Suivi des loyers |
| **Maintenance** | Demandes de réparation |
| **Rapports** | Revenus et performances |

## Sécurité

### Activer l'Authentification à Deux Facteurs
1. Allez dans **Paramètres** → **Sécurité**
2. Cliquez sur **Activer 2FA**
3. Scannez le QR code avec votre app (Google Authenticator, Authy)
4. Entrez le code à 6 chiffres pour confirmer

## Support

Pour toute question, contactez votre administrateur ou consultez les rapports pour analyser vos données.
